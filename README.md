Senior backend developer with **10+ years** helping startups across the US, Europe, and the Middle East replace manual workflows with systems built for real production environments.

- **AI integration & APIs** — OpenAI, Claude, Gemini, and Groq inside Laravel apps: chatbots, content generation, smart search, document Q&A, RAG pipelines
